# Prüfschema (Workshop-Demo)

## Prüfschritte
1. Falldaten auf Vollständigkeit prüfen
2. Produktname und Kontext erfassen
3. Relevante Regeln identifizieren
4. Vorprüfung mit Begründung durchführen
5. Unsicherheiten / offene Punkte dokumentieren
6. Optional: Entwurf Abschlussschreiben erstellen

## Pflichtfelder (mindestens)
- Antrag-ID
- Produktname
- Wirkstoff
- Verfahrensart
- Antragsteller
- Sprachversion
