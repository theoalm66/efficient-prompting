# Vorlage Abschlussschreiben (Entwurf)

## Struktur
1. Betreff
2. Bezug / Antrag
3. Ergebnisdarstellung (als Entwurf)
4. Begründung (wesentliche Gründe)
5. Hinweise / offene Punkte
6. Schlussformel

## Stil
- neutral
- formal
- nachvollziehbar
- keine finale Rechtsentscheidung simulieren
